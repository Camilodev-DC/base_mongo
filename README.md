Estudiantes de Ciencia de Datos - Registro General
📖 Descripción del Proyecto
Este proyecto proporciona una estructura de datos estandarizada en Python que contiene un registro de 30 estudiantes. Todos los registros han sido normalizados para reflejar una edad uniforme y pertenencia al programa académico de Ciencia de Datos.

El objetivo es ofrecer un conjunto de datos limpio y listo para ser utilizado en tareas de manipulación de datos, pruebas de algoritmos de búsqueda/ordenación, o como base para demostraciones de programación.

🛠️ Estructura de Datos (Python)
Los datos están organizados como una Lista de Listas. Esta estructura permite acceder a cada registro individual (una sublista) y, a su vez, a los campos específicos de cada estudiante mediante índices.
